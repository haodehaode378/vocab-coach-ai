# AI Agent 记单词项目｜01 项目总览（MVP）

## 项目目标
- 在 2 周内交付可内测版本：支持导词、学习、复习、统计、AI 助教。
- 聚焦英语词汇记忆场景，优先保证学习闭环可跑通。

## 用户画像
- 英语学习者（四六级、考研、职场）
- 需要高频复习和弱项强化

## MVP 范围
- 词库管理：手动新增、CSV 导入、标签管理
- 学习卡片：单词、音标、释义、例句、记忆提示
- SRS 复习：SM-2 算法，Again/Hard/Good/Easy 评分
- 练习模式：选择题 + 拼写题
- 学习统计：今日学习、今日复习、正确率、连续天数
- AI 助教：解释词义差异、生成小测、给学习建议

## 非目标（本期不做）
- 复杂多 Agent 协作编排
- 多端离线同步
- 语音打分与发音矫正
